Array Methods

map,find,filter,reduce,slice,splice,indexof,include(string Array) ,flat, flarmap ,reverse ,at, sort ,tosort

**MAP-**Map is used to transform an array
1. map()
Used to create a new array by applying a function to each element.
Does not modify the original array.

2. find()
Used to find the first value that satisfies a given condition.
Returns one value, not an array.
Returns undefined if not found.

3. filter()
Used to filter values based on a condition.
Returns a new array of all matching values.

4. reduce()
Used to reduce array values into a single value (sum, product, merge, count, etc.).
Works using a callback accumulator.

5. slice()
Used to copy a portion of an array.
Returns a new array.
Does not change the original array.

6. splice()
Used to add, remove, or replace elements in an array.
Modifies the original array.

7. indexOf()
Used to find the index of a value.
Returns -1 if the value is not found.

8. includes()
Used to check whether a value exists in an array or string.
Returns true/false.

9. flat()
Used to flatten nested arrays into a single-level array.

10. flatMap()
Combination of map() + flat() (one-level flat).
Used to transform and then flatten the result.

11. reverse()
Used to reverse the order of array items.
Modifies the original array.

12. at()
Used to get a value by index, supports negative index.
Example: arr.at(-1) gives the last element.

13. sort()
Used to sort array values.
Converts elements to strings by default.
Modifies the original array.
Custom sorting can be done using a compare function.

14. toSorted()
New method.
Same as sort() but does not modify the original array.
Returns a new sorted array.

API-application programming interface

-->In JavaScript, an API is a set of tools or services that your JavaScript code can use to communicate with something else.

backend data > json(array of object) >  

Types of APIs in JavaScript:

1. Browser APIs 

These are built into the browser.

Examples:

DOM API → lets JS control HTML elements
Fetch API → used to call servers (get data)
LocalStorage API → store data in browser
Geolocation API → get user's location
Canvas API → draw graphics

Example:
 document.getElementById("btn"); // DOM API
2. Web APIs (Server APIs)

These are APIs created by websites/servers that provide data.

Example:
Calling a weather API to get temperature.

fetch("https://api.weather.com/data")
    .then(res => res.json())
    .then(data => console.log(data));

Here, fetch() sends a request → API sends back data → JS uses it.

Synchronous:
JavaScript runs one task at a time, in order, line by line.
It waits for one task to finish before starting the next one.
✔ Sync = Blocking → if one line takes time, everything waits.

Asynchronous:
JavaScript can start a task and continue with the next task without waiting.
Async code will finish later, in the background.

Example (Async using setTimeout):
console.log("Start");

setTimeout(() => {
    console.log("Async Task");
}, 2000);

console.log("End");

await?
is a keyword used inside async functions to wait for a Promise to finish.
It pauses the function until the Promise returns a result.
