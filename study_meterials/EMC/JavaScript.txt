A JavaScript function is the block of code designed to perform particular task.

in JavaScript, the return keyword is used within a function to specify the value that the function should produced as its result or return to the caller. When a function is executed and encounters a return statements ,it immediately stops the value and return the specified value to the calling code.

In JavaScript for loop is a control flow statement used to repeatedly execute a block of code based on specified condition:
syntax:

}
<-------------->
